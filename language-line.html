Language Line - CSA Q&A Guide

What is Language Line?

Language Line is a telephone interpreting service used to help communicate with customers who need information in their own language. The service operates using a three-way phone call.

What number do I call for Language Line?

You should call 0845 310 9900 to reach Language Line.

What ID code do I need to use?

When prompted, quote Paradigm’s ID code: 700174.

How do I set up a Language Line call?

  1. Call Language Line on 0845 3109900.
  2. Provide Paradigm’s ID code: 700174.
  3. Explain the nature of the call to the operator.

What if the resident is not with me?

Let the operator know whether the resident is present. If a dial-out is required, you will remain on the line while the operator first connects the translator, and then calls the resident.

Do I need to set up the conference call myself?

No, Language Line arranges the conference call at their end. You only need to stay on the line and follow the operator’s guidance.
